Enum EConnectStatus#

  • Defined in File ipc.h

Enum Documentation#

enum class sdv::ipc::EConnectStatus : uint32_t

Connection status enumeration.

Values:

enumerator uninitialized

Initialization via IConnect pending/required.

enumerator initializing

Initiation channel access.

enumerator initialized

Channel access initialized.

enumerator connecting

Both participants are available, connecting started.

enumerator negotiating

Negotiating the connection.

enumerator connection_error

Connection error; negotiation failed.

enumerator connected

Connection successful.

enumerator communication_error

Communication error.

enumerator disconnected

Connection terminated due to disconnection of other participant.

enumerator disconnected_forced

Connection terminated due to unplanned disconnection of other participant.

enumerator terminating

Termination of channel access (through IObjectDestroy::DestroyObject)
