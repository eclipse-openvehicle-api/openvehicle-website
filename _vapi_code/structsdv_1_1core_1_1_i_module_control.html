struct IModuleControl#

Interface for module control.

Attention

This interface is accessible by standalone and essential apps.

Subclassed by CModuleControl

Public Functions

virtual TModuleID Load(const sdv::u8string &ssModulePath) = 0#

Load a module with the provided path containing SDV components. The components are available for creation through the repository.

This method can be used to load a module by providing a path to SDV components. Upon successful loading it will return the ID of the module. If the module Path is wrong/invalid or empty, module cannot not be loaded and 0 is returned. If the module is successfully loaded once and same module is loaded again the same module ID is returned.

Attention

Only modules with exposed SDV functions and the correct interface version can be loaded.

Parameters:

ssModulePath – [in] Path to the module to load.

Returns:

Returns the ID of the module or 0 if the module could not be loaded.

virtual bool Unload(TModuleID tModuleID) = 0#

Unload the module by providing module ID. The function will request the repository to shutdown any running component. If there are no components running any more, the module will be unloaded.

Parameters:

tModuleID – [in] Id representing a module, which has previous been loaded by Load

Returns:

Returns ‘true’ when unloading was successful or the module was unloaded before already. Returns ‘false’ in case the module could not be unloaded due to having active objects.

virtual bool HasActiveObjects(TModuleID tModuleID) const = 0#

Checks for active objects - if active objects are present in a module, the module (currently) cannot be unloaded via Unload.

The module exposes the symbol HasActiveObjects defined in export/support/component_impl.h, which is to be used in order to determine whether or not active objects remain within the module

Parameters:

tModuleID – [in] Id representing a module, which has previous been loaded by Load(ModulePath)

Returns:

Returns true if the module is currently loaded and it contains at least one loaded object and false otherwise.
