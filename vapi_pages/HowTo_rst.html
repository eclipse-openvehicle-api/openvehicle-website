How to write reStructuredText#

To write an .rst (reStructuredText) file, follow these instructions:

  1. File Extension: Save the file with a .rst extension, e.g., example.rst.

  2. Headings: Use = for the main title, - for subheadings, and ~ or ^ for deeper levels. Ensure the underline matches the title length.

    Here's a widely used and recommended order:

Level   Character       Example
H1      =       Main Title
H2      -       Major Section
H3      ~       Subsection
H4      ^       Sub-subsection
H5      "       Lower-level section
H6      '       Even lower-level

Title1#

Title2#

Title3#

Title4#

Title5#
Title6#

  1. Paragraphs: Write plain text for paragraphs. Leave a blank line between paragraphs.

  2. Lists:

    • Use - or * for unordered lists.

    • Use numbers followed by a period for ordered lists.

    - Item 1
- Item 2

1. First
2. Second

  3. Links: Use backticks and an underscore for hyperlinks.

    `Link Text <https://example.com>`_
  4. Inline Formatting:

    • Bold: **text**

    • Italic: *text*

    • Literal text: Use double backticks.

  5. Images: Use the .. image:: directive.

    .. image:: path/to/image.png
      :alt: Description
      :width: 300px

  6. Tables: Use the .. table:: or .. csv-table:: directive.

    .. table:: Example Table

      +------+-----+
      | Col1 | Col2|
      +------+-----+
      | Val1 | Val2|
      +------+-----+
    Example Table#

    Col1

    Col2

    Val1

    Val2

  7. Code Blocks: Use the .. code-block:: directive.

    .. code-block:: cpp

      #include <iostream>

      void hello()
      {
        std::cout << "Open Vehicle API!" << std::endl;
      }

  8. Admonitions: Use directives like .. note::, .. warning::, or .. tip::.

    .. note::

       This is a note.

  9. Figures: Use the .. figure:: directive for images with captions.

    .. figure:: path/to/image.png
       :scale: 50%

       Caption text.

  10. Include Files: Use the .. literalinclude:: directive to include external files.

    .. literalinclude:: path/to/file.cpp
       :language: cpp

  11. Epigraphs: Use the .. epigraph:: directive for quotes.

    .. epigraph::
       "To be or not to be."
       :attribution: Shakespeare

    “To be or not to be.” :attribution: Shakespeare

  12. TabSets: Use the .. tab-set:: and .. tab-item:: directives for tabbed content.

    .. tab-set::

       .. tab-item:: Tab 1

               Content for Tab 1.

       .. tab-item:: Tab 2

               Content for Tab 2.

    Content for Tab 1.

    Content for Tab 2.

  1. Tabs: Use the .. tabs:: and .. tab:: directives for tabbed content.

    .. tabs::

    .. tab:: Tab 1
        Content for Tab 1 goes here.

    .. tab:: Tab 2
        Content for Tab 2 goes here.

    .. tab:: Tab 3
        Content for Tab 3 goes here.

    Content for Tab 1 goes here.

    Content for Tab 2 goes here.

    Content for Tab 3 goes here.

  2. Cards: Use the .. card:: directive for card-like content.

    .. card:: Card Title

       Header
       ^^^

       Content
       +++

       Footer

    Header

    Card Title

    Content

    Footer

  3. seealso: Use the .. seealso::

    .. seealso:: This is a simple **seealso** note.

    See also

    This is a simple seealso note.

  4. note: Use the .. note::

    .. note::  This is a **note** box.

    Note

    This is a note box.

  5. warning: Use the .. warning::

    .. warning:: This is a **warning** box.

    Warning

    This is a warning box.

  6. tip: Use the .. tip::

    .. tip:: This is a **tip** box.

    Tip

    This is a tip box.

  7. attention: Use the .. attention::

    .. attention:: This is a **attention** box.

    Attention

    This is a attention box.

  8. caution: Use the .. caution::

    .. caution:: This is a **caution** box.

    Caution

    This is a caution box.

  9. hint: Use the .. hint::

.. hint:: This is a **hint** box.

Hint

This is a hint box.

  1. important: Use the .. important::

.. important:: This is a **important** box.

Important

This is a important box.

  1. error: Use the .. error::

.. error:: This is a **ierror** box.

Error

This is a ierror box.

  1. todo: Use the .. todo::

.. todo:: This is a **todo** box.

  1. danger: Use the .. danger::

.. danger:: This is a **danger** box.

Danger

This is a danger box.

These are the basic elements for writing .rst files. Use them as needed for your documentation.
