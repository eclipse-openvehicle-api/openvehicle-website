Example Interface Set
=====================

.. note::
   An :term:`Interface Set` is just a collection of :term:`Interface` s. These may not be from the same developer or company as long as each interface name is unique within the set. Also data structures may be described in an Interface Set. An :term:`Interface Set` can be composed of multiple header files. For components to be developed for a system the :term:`Interface Set` of the system has to be known.

.. uml::
   :caption: Interface of Example reception signal

   @startuml
      set separator ::
      Interface vss::Device::IReceptionSignalSpeed_Event
      {
         virtual void SetSpeedValue(int32_t value)
      }
      Interface vss::Device::IReceptionSignalSpeed
      {
         virtual void RegisterSpeedEvent(IReceptionSignalSpeed_Event* event)
         virtual void UnRegisterSpeedEvent(IReceptionSignalSpeed_Event* event)
      }

      Interface vss::Service::IReceptionSignalSpeed
      {
         virtual int32_t GetSpeedValue()
         virtual void RegisterCallBack(vss::Device::IReceptionSignalSpeed_Event* callback) 
         virtual void UnregisterCallBack(vss::Device::IReceptionSignalSpeed_Event* callback) 
      }
   @enduml
