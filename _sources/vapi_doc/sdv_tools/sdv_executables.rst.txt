List of Executables
-------------------

The executable files of the framework can be assigned to different areas.


Utilities (Auto Code Generation)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- sdv_idl_compiler
- sdv_dbc_util
- sdv_vss_util

   Utilities for auto generating code of Proxy & Stub implementations for marshalling, vehicle devices, basic services, data link components and FMUs for simulating.

Packaging and Installing
~~~~~~~~~~~~~~~~~~~~~~~~

- sdv_packager

   Executable for creating and/or installing software packages.

System Control
~~~~~~~~~~~~~~

- sdv_control
- sdv_trace_mon
- sdv_local_shutdown

   Maintaining the system, start & stop of Open Vehicle API instances, monitoring the system, installing and deinstalling software packages, starting and stopping services.
 
System Internals
~~~~~~~~~~~~~~~~

- sdv_core
- sdv_iso

   These are internal executables which are not intended for user interaction.
